# OSX Setup

## Terminus font

```sh
brew install --cask font-terminus
```

Download the nerd font version from here: `https://www.nerdfonts.com/font-downloads`

## Scrollwheel fix

```sh
brew install --cask unnaturalscrollwheels
```

## Window tiling

```sh
brew install rectangle
```

## LazyVim

```sh
git clone https://github.com/LazyVim/starter ~/.config/nvim
brew install rg
brew install fd
```

## iTerm config

- Set the colors in the `profile`
- Set the theme to 'minimal' in `appearance/general`
- Set the font to Terminus (TTF) in the `profile/text`

## Zsh

Apply the config

## Colors

git clone https://github.com/dracula/iterm.git
